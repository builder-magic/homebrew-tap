# Homebrew Tap

Homebrew formulae for [builder-magic](https://github.com/builder-magic) projects.

## Usage

```bash
brew tap builder-magic/tap
brew install <formula>
```

Or install directly:

```bash
brew install builder-magic/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `timebound-iam` | Temporary AWS credential broker that runs as an MCP server |
